## ğŸ“ SUPPORT & MAINTENANCE

### Adding New Translation Keys:

1. Open `lib/utils/language_provider.dart`
2. Add key to English section:
   ```dart
   'new_key': 'New Text',
   ```
3. Add same key to all other languages
4. Use in code:
   ```dart
   Text(context.tr('new_key'))
   ```

### Adding New Language:

1. Add to `_languages` map in language_provider.dart
2. Add new translation section
3. Copy all keys from English
4. Translate to new language
5. Done!

### Fixing Translation:

1. Find the key in language_provider.dart
2. Update the text for that language
3. Save file
4. Hot reload - changes appear immediately
